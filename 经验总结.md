# mac上启动docker的问题

Docker 守护进程未运行。在 macOS 上，您需要启动 Docker Desktop 应用。

但更简单的方法是使用本地部署，而不是 Docker：

这会直接启动前后端服务，无需 Docker。之前我们已经配置好了 conda 环境和所有依赖，系统应该能正常运行。

如果您必须用 Docker，需要：

```markdown
- 打开 Applications 文件夹
- 双击 Docker.app 启动 Docker Desktop
- 等待 Docker 启动完成（菜单栏会显示 Docker 图标）
- 然后再运行 docker-compose up -d --build
```
